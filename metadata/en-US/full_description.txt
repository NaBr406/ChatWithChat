ChatWithChat is a local-first Android chat assistant for working with multiple AI providers from one app.

Supported providers:
- OpenAI
- Anthropic Claude
- Google Gemini
- Groq
- Ollama
- OpenRouter
- Custom OpenAI-compatible endpoints

Highlights:
- Start chatting from the first screen.
- Switch models from the chat top bar.
- Keep chat history in a side drawer with search, duplicate, and delete actions.
- Configure API URL, API key, model, system prompt, temperature, top p, and streaming timeout per provider.
- Fetch available model IDs from supported provider URLs.
- Compare answers from multiple providers in one conversation.
- Manage long-term personal memories locally, with review, edit, archive, delete, and Markdown export.
- Edit and retry messages, keep assistant answer revisions, and export chat history.
- Attach supported images with size checks and large-image compression before upload.

Privacy:
Chat history, provider settings, API keys, available model lists, and memories are stored on your device. The app sends chat content only to the AI service endpoints you configure.

Note:
Some providers, models, or APIs may be unavailable in certain regions or accounts. API usage may require billing with the provider.
